Reversing a Queue Open on JetBrains Academy

Suppose queue is a pre-defined queue. Define reversed_queue, a queue that contains the elements of the initial_queue in the reversed order.

For example, if queue contains elements 1, 2 and 3, then reversed_queue should contain elements 3, 2, 1.

queue is defined as a deque and is ready to use. Note that appendleft() was used to add new elements to it, so the head of the queue is on the right.

You don't need to print out the resulting queue.
